VOLUNTEER APP UI DESIGN

Volunteer Me!

UI design for the volunteer app. The feature of this app is that users can find the volunteer opportunites from each volunteer organization. Users will be able to find it by using a calendar system that syncs to your phone, or using a search system. The idea of this app is originally made for the group project at BCIT and I was in charge of UI Design. Using warm colour scheme and simplified icons create friendly and welcoming atmosphere.
